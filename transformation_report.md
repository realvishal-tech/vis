## 🎓 Customization Guide

### Change Primary Color
Edit `modern-theme.css`:
```css
:root {
  --neon-cyan: #your-color;
}
```

### Disable Code Rain
In `modern-effects.js`, comment out:
```javascript
// new CodeRainGenerator();
```

### Adjust Animation Speed
Modify transition duration:
```css
--transition: 0.5s cubic-bezier(...); /* Change 0.5s */
```

### Add Custom Toast
Use the Toast class anywhere:
```javascript
Toast.success('Custom message here!');
```

## 🐛 Troubleshooting

**Problem**: Animations not appearing?
- **Solution**: Clear cache (Ctrl+Shift+R / Cmd+Shift+R)

**Problem**: Cursor glow not showing?
- **Solution**: It only shows on desktop/laptop, not mobile

**Problem**: Code rain too intense?
- **Solution**: Edit opacity in modern-effects.js

**Problem**: Mobile feels laggy?
- **Solution**: This is normal on older devices; effects are optimized
