# BCAverse AI - Quick Setup Guide

## 🚀 What is BCAverse AI?

BCAverse AI is an intelligent, context-aware assistant designed specifically for BCA students. It helps with:
- 💻 Coding problems and explanations
- 📚 Course concepts and theory
- 🐛 Debugging and troubleshooting
- 📝 Study notes generation
- 🚀 Career guidance and advice

## 📱 Getting Started

### Step 1: Access BCAverse AI
```
1. Go to main website (index.html)
2. Click "🤖 AI Assistant" in navbar
3. Start chatting - no setup required!
```

### Step 2: Try Quick Actions
```
📖 Explain - Concept explanations
💻 Code - Generate code samples
📝 Notes - Create study notes
🚀 Career - Career advice
```

### Step 3: Customize Settings (Optional)
```
1. Click ⚙️ Settings in sidebar
2. Adjust your preferences
3. Add OpenAI API key (optional for GPT-4)
```

## ⚙️ Configuration

### Option 1: Use Local AI (Default - No Setup)
- ✅ Works instantly
- ✅ No API key needed
- ✅ Intelligent responses from built-in knowledge
- ⚠️ No internet API integration

### Option 2: Add OpenAI API Key (Optional)
1. Get key from https://platform.openai.com/api-keys
2. Open bcaverse-ai-settings.html
3. Paste key and save
4. Enjoy GPT-4 responses

## 🔧 Troubleshooting

### Chat Not Responding?
```bash
1. Press F12 to open Developer Tools
2. Check Console tab for errors
3. Ensure JavaScript is enabled
4. Try clearing browser cache
```

### Code Not Highlighted?
```
- Check code block format: ```language code```
- Ensure Highlight.js loads from CDN
- Try manual language specification
```

### Lost Chat History?
```
- Check if localStorage is enabled
- Don't clear browser data
- Chat exports to JSON when requested
```

## 🔐 Safety & Privacy
